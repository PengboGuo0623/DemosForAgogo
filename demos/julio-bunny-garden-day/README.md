# Julio's Garden Day

Julio's Garden Day is a landscape-only H5 light interactive demo for the Julio Bunny preschool series.

## Demo Entry

- `index.html`: static GitHub Pages entry.
- `assets/`: built JS, CSS, and sprite assets.
- `source/`: Vite + TypeScript + PixiJS + GSAP source project.

## Current Scope

- iPhone 14 Pro Max landscape target: `932 x 430` CSS px.
- Portrait rotation prompt.
- Play, tap-sun, plant flowers, water flowers, collect carrots, finale, and replay.
- Programmatic animation feedback for Julio, water, flowers, carrots, clouds, and celebration.
- No music, sound effects, App SDK, login, sharing, or analytics.

## Source Scripts

From `source/`:

```sh
npm install
npm run dev
npm run build
```
